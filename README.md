# Extrokids Toy Store — Website

A responsive toy-store website for **Extrokids**, built as a single self-contained HTML file (no build step, no dependencies).

- **`index.html`** — *Extrokids Arcade* (neon gaming-console theme: 8-bit loader, synthwave hero, spin-the-wheel, Konami code, filterable catalog).
- **`showroom.html`** — *Premium Toy Store* (warm retail-showroom theme: illustrated SVG store interior, wooden display cases).

> Want the showroom as the homepage instead? Just swap the filenames: rename `index.html` → `arcade.html` and `showroom.html` → `index.html`.

Real Extrokids logo and real product photos (loaded live from Extrokids' CDN) are used throughout. Product images require an internet connection to display.

## Deploy on GitHub Pages (free, ~1 min)
1. Go to **github.com** → **New repository** → name it `extrokids` → **Public** → **Create repository**.
2. On the repo page click **"uploading an existing file"** and drag in `index.html`, `showroom.html`, and this `README.md`. Click **Commit changes**.
3. Open **Settings → Pages**.
4. Under **Source**, pick **Deploy from a branch**, set branch to **`main`** and folder **`/ (root)`**, then **Save**.
5. Wait ~1 minute, refresh, and your live link appears:
   **`https://YOUR-USERNAME.github.io/extrokids/`**

## Even faster (no Git needed)
Drag this whole folder onto **app.netlify.com/drop** — you'll get an instant live URL.
